# AI Personal Assistant - User Manual

## Getting Started

1. **Sign Up / Login**
   - Use Google, Apple, Microsoft, or Email + Passkey
   - Enable MFA for maximum security

2. **First Conversation**
   - Open the app and start chatting naturally
   - Try voice by tapping the microphone

3. **Permissions**
   - The app will ask for camera, microphone, and screen access
   - You can grant/revoke anytime in Settings → Privacy

## Key Features

### Voice Conversation
- Say "Hey Assistant" (optional wake word)
- Interrupt the AI while it's speaking
- Switch between multiple languages

### Screen Sharing
1. Click **Share Screen**
2. Select the window or entire screen
3. Ask questions like:
   - "What should I click next?"
   - "I'm stuck on this error"
   - "Teach me how to use this tool"

### Learning Mode
- Open Photoshop / VS Code / Figma
- Share screen
- Say: "Teach me how to create a gradient in Figma"

### Memory
- Memory is **off by default**
- Say: "Remember that I prefer concise answers"
- Manage memories in the Memory tab

### Agents
- Switch agents in the chat header:
  - **Coding Agent**
  - **Learning Agent**
  - **Research Agent**
  - etc.

## Privacy & Security

- All memory is encrypted
- You control every permission
- Export or delete your data anytime
- Remote logout available from any device

## Advanced Usage

- **YouTube**: Paste a link and ask questions
- **Documents**: Upload PDFs and ask anything
- **Camera** (mobile): Point camera at hardware or documents for live help

## Troubleshooting

- Voice not working? Check microphone permission
- Screen sharing slow? Use wired connection or reduce quality
- Need offline mode? Enable "Local Models" in settings
